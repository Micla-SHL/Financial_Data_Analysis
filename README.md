# 金融数据分析学习指南

## 项目概述
本项目旨在帮助初学者掌握 **SQL、Pandas 和知识图谱** 三大工具在金融关系挖掘和智能分析中的应用。

## 学习目标
- 掌握SQL在银行数据分析中的应用
- 熟练使用Pandas进行金融数据处理和分析
- 理解知识图谱在金融关系挖掘中的作用
- 具备金融风险识别和反欺诈分析能力

## 学习路径

### 阶段一：基础理论（1-2周）
1. **[SQL基础知识](01_SQL基础知识.md)**
   - 基础语法和查询
   - 银行数据结构设计
   - 复杂查询和聚合分析

2. **[Pandas基础知识](02_Pandas基础知识.md)**
   - 数据结构和基本操作
   - 数据清洗和预处理
   - 金融数据处理技巧

3. **[知识图谱基础知识](03_知识图谱基础知识.md)**
   - 图数据库概念
   - 金融知识图谱构建
   - 关系挖掘和风险分析

### 阶段二：实战项目（2-3周）
4. **[实战练习项目](04_实战练习项目.md)**
   - 完整的银行数据分析项目
   - SQL+Pandas+知识图谱综合应用
   - 自动化报告生成

## 学习建议

### 1. 学习顺序
- 先掌握SQL基础，再学习Pandas
- 理解理论知识后立即动手实践
- 每个阶段完成对应的练习题目

### 2. 实践方法
- 运行所有示例代码
- 修改参数观察结果变化
- 用自己的数据进行分析
- 尝试解决实际问题

### 3. 重点难点
- **SQL**：复杂JOIN操作、窗口函数
- **Pandas**：时间序列处理、聚合分析
- **知识图谱**：图算法、关系挖掘
